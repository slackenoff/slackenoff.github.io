    NAS

    NAS， Network attached storage，是一种用于网络存储设备的通讯协议，允许用户通过局域网或互联网连接到存储设备并访问和管理其中的数据。NAS有多种实现方式，包括NFS、CIFS/SMB、FTP、HTTP等。

    NAS协议

    是一种基于文件级别的存储方案，可以提供高效的数据共享和备份，并支持多用户同时访问。

可以在各种操作系统上运行，如linux、windows、unix以及macos

    实现

    NFS
    
    NETWROK FILE SYSTEM，基于UDP协议，主要用于unix/linux系统； 跨平台，允许不同操作系统的计算机通过网络共享文件和目录。

    工作原理：
            1. 客户端发请求给服务端，请求读取或写入文件或目录
            2. 服务器响应请求并返回请求的数据或执行请求的操作
            3. 如果请求，需要传输数据，则数据通过网络传输
            4. 数据缓存，客户端将最近访问的数据存储在本地缓存中，以便以后更快地访问相同地数据
            5.锁定机制，当多个客户端同时访问一个文件时，NFS提供锁定机制来保证数据地完整性和一致性

CIFS/SMB

    Common Internet File System ， 基于tcp/ip协议，主要用于windows系统

AFP

    apple filing protocol  

FTP

    file transfer protocol 

    ASCII模式（传输纯文本文件）和二进制模式(传输图像、视频和音频文件)

    不安全协议，替代协议 FTPS和SFTP

    FTP://访问服务站点

    工作机制： 上传与下载， 先获取远程文件的副本，然后对副本进行修改，最后将修改后的文件副本上传服务器

WEBDAV

     web distributed authoring and  versioning ，一种基于http协议的文件管理协议，主要用于web应用

ISCI

    Internet small computer system interface ， 基于IP协议的存储网络协议，可以将远程存储设备映射位本地磁盘

    特点/优越性

    NAS使得存储设备变得更加灵活、可靠、高效，并提供更好的数据分享和备份功能。

安全性，数据传输加密，保证数据的完整性和机密性

可靠性，嫩巩固快速识别故障并自动切换到备份设备

传输效率较高（缓存、预读等）

跨平台，支持多种操作系统和应用程序
